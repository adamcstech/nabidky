# Nabídky — generátor nabídek a smluv

Interní systém **CS Technologies** pro generování klientských **nabídek** a **smluv**
jako statických stránek na nehádatelných **GUID URL** (`https://nabidky.cstech.cz/<guid>`).

## Brandy
- **CS Technologies s.r.o.** — výroba webů, eshopů, custom webová řešení
- **CS Bot** — AI voiceboti, automatizace procesů, AI agenti
- **Samolepák.cz** — eshop, 3D samolepky / logoprinty *(provozuje CS Production Group s.r.o.)*

Nabídky pro všechny tři brandy; smlouvy pouze CS Tech + CS Bot.

## Stav
MVP / fáze 1 — engine + šablony (vzor: rebrand nabídky SoftMedia).
Docroot = `www/`, generované nabídky → `www/<guid>/`.
