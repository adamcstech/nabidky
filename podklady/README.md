# 📥 Podklady (intake)

Sem nahraj všechny materiály. Frank si je **nejdřív celé prostuduje**, teprve pak stavíme.
Tahle složka je **mimo git** (jen tenhle README se verzuje) — můžou tu být citlivá data.

## Kam co patří

Pro každý brand (`cs-technologies`, `cs-bot`, `samolepak`):

| Složka | Co tam dát |
|---|---|
| `01-vzorove-nabidky/` | **Vzorové nabídky** — to dobré, čím se chceme inspirovat (klidně i SoftMedia apod.) |
| `02-soucasne-odesilane/` | **Co posíláme teď** — i to „hnusné". Ať vím, z čeho jdeme a co nahrazujeme |
| `03-loga-a-podklady/` | **Loga, brand manuál, fonty, barvy, fotky, ceníky, texty** |

`_spolecne/` = materiály napříč brandy (sdílené šablony, vzory, právní subjekty…).

## Formáty
Cokoliv — PDF, DOCX, PNG/SVG, .eml/.txt e-maily, screenshoty. Klidně to tam jen naval, srovnám si to sám.

## Až bude nahráno
Napiš **„prostuduj"** (nebo „jedem") a Frank:
1. projde všechny materiály a udělá si shrnutí (styl, struktura, brand prvky),
2. teprve potom postaví demo nabídku ve stylu SoftMedia v našem brandu.
